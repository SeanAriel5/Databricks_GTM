AWS Partnership
Databricks on AWS is the platform's foundational multi-cloud deployment — with deep S3, IAM, and SageMaker integrations, a flagship marketplace listing, and a growing mainframe modernization play with AWS Transform.
AWS Amazon S3 HyperScaler Lakebase on AWS Mainframe Modernization
Partnership Overview

Databricks was born on AWS — Amazon S3 was the original cloud object store underpinning the first Lakehouse deployments. Today, AWS remains Databricks' highest-volume cloud, with the largest installed base of enterprise customers and the deepest catalog of first-party integrations. The AWS Marketplace listing makes Databricks purchasable against existing AWS Enterprise Discount Program (EDP) commitments, creating a natural commercial on-ramp for AWS-centric enterprises. In 2026, the partnership deepened significantly through a shared initiative with AWS Transform for AI-powered mainframe modernization — a high-value play that positions Databricks and Accenture jointly for legacy migration engagements.

2026 Key Milestones on AWS
  • Lakebase GA on AWS — Serverless PostgreSQL-compatible OLTP database (Lakebase Autoscaling) is now generally available on AWS Databricks. Provisions in seconds, scales to zero automatically, and exports OpenTelemetry traces to AWS CloudWatch for full observability of OLTP workloads alongside Lakehouse analytics.
  • AWS Transform Mainframe Partnership — Databricks and AWS co-launched a coalition with Accenture, Capgemini, and IBM to use agentic AI and AWS Transform for composable mainframe modernization. AI agents extract PL/I business logic, generate modern equivalents, and store results on the Databricks Lakehouse for testing and governance.
  • April 2026 Release: GPT-5.5 + Lakeflow Designer — Major AWS-specific release notes including support for GPT-5.5 models via AI Gateway, the GA of Lakeflow Designer (drag-and-drop pipeline builder), and OpenTelemetry export from Lakebase to AWS CloudWatch — enabling full observability without leaving the AWS ecosystem.
  • Lakebase Provisioned Retirement — AWS documentation indicates the older Lakebase Provisioned model is being superseded by Lakebase Autoscaling — which scales to zero, eliminating idle costs and removing the need for manual instance sizing. Customers on Provisioned are being migrated to Autoscaling.
Key AWS-Native Integrations
Amazon S3 & Lake Formation
Delta Lake tables stored natively in S3, with Unity Catalog governing access alongside AWS Lake Formation permissions. Customers can grant fine-grained row/column access through Unity Catalog while maintaining S3-bucket-level policies in IAM — both enforced at query time.
AWS Marketplace & EDP
Databricks is listed on the AWS Marketplace with EDP-eligible billing. Enterprise customers can apply existing AWS spending commitments to Databricks consumption, making procurement frictionless and enabling commercial teams to bundle Databricks into AWS EDP negotiations.
SageMaker Integration
MLflow models registered in Unity Catalog can be deployed to SageMaker endpoints, bridging Databricks' experimentation environment with SageMaker's real-time inference infrastructure. Data scientists can train on Databricks and serve on SageMaker without data egress or governance gaps.
AWS CloudWatch & OpenTelemetry
Lakebase (April 2026) now exports OpenTelemetry traces to AWS CloudWatch, enabling unified monitoring of OLTP query performance alongside Lakehouse metrics. Operations teams can maintain a single AWS-native observability plane without additional tooling.

Mainframe Modernization
AI-Powered Legacy Migration with AWS Transform
Accenture, Databricks, and AWS jointly modernize mainframe workloads: AI agents extract PL/I / COBOL business logic, AWS Transform generates cloud-native equivalents, and Databricks provides the governed data foundation for testing and validation. Eliminates $370M in annual technical debt for typical enterprise clients.
Financial Services
Risk & Compliance on AWS Lakehouse
FSI clients (Morgan Stanley, Rabobank) run BCBS 239 data lineage, market risk models, and fraud detection on Databricks Lakehouse on AWS. Unity Catalog enforces FINRA and SEC data governance automatically, while Delta Lake provides the time travel needed for regulatory audit trails.
Supply Chain
AI-Powered Supply Chain Orchestration
Kinaxis Maestro now integrates with Databricks on AWS via Delta Sharing, enabling real-time supply chain data from Databricks Lakeflows to feed directly into Kinaxis demand sensing and multi-tier planning without data duplication or egress fees.
Healthcare & Life Sciences
Clinical Data Lakehouses on AWS
Healthcare organizations leverage AWS HIPAA-eligible services with Unity Catalog's PHI access controls to unify EMR, claims, and genomics data. Agent Bricks is being piloted for clinical trial data querying and adverse event detection across structured and unstructured medical records.
