Certification Overview

The Databricks Certified Machine Learning Associate validates foundational competence in building, tracking, and deploying machine learning models on the Databricks Lakehouse. While the Data Engineer certifications focus on pipeline architecture and data reliability, the ML Associate targets the data scientist and ML engineer persona — validating knowledge of the full model lifecycle from exploratory analysis through production deployment. This is the recommended first certification for Accenture's AI/ML consulting teams and client-facing data scientists working on Databricks engagements.

Format: 120 minutes · ~60 questions · Multiple-choice and scenario-based · Recertification required every 2 years · No formal prerequisites, though hands-on Databricks experience is assumed.

29%
Databricks ML — Core Tools & AutoML
Databricks AutoML (classification, regression, forecasting), Databricks Notebooks for ML (cluster configuration, GPU clusters, collaborative development), integration of Spark MLlib vs. single-node scikit-learn on Databricks, and the Databricks ML runtime (preinstalled libraries, GPU support).
Foundational
33%
ML Workflow — MLflow Experiment Tracking
MLflow runs and experiments (mlflow.start_run(), logging params/metrics/artifacts), autologging with mlflow.autolog(), comparing runs in the MLflow UI, tracking hyperparameter tuning with Hyperopt, model signatures and input example logging, and experiment sharing across workspaces.
Core
19%
ML Workflow — Feature Store
Creating and updating feature tables, feature lookups for training and serving, point-in-time correct feature retrieval, Unity Catalog integration for feature governance, and the relationship between Feature Store and training vs. serving data pipelines to prevent training-serving skew.
Core
19%
ML Workflow — Model Registry & Production Deployment
Registering models in Unity Catalog Model Registry (mlflow.register_model()), model versioning and lifecycle stages (Staging/Production/Archived), model serving endpoints (real-time and batch), Databricks Model Serving autoscaling, inference tables for monitoring, and A/B testing with champion/challenger routing.

2026 Exam Updates
  • Unity Catalog Model Registry replaces workspace registry — All model registration questions now reference the Unity Catalog-backed registry (not the deprecated workspace MLflow registry). Candidates must understand the 3-level namespace (catalog.schema.model) for model registration.
  • GenAI evaluation is now testable — The exam includes questions on using MLflow 3's LLM-as-a-judge evaluation for generative models — correctness scoring, safety evaluation, and human feedback loops. Reflects Databricks' strategic push into LLMOps.
  • Feature Store + Delta Lake integration — Updated questions test understanding of how feature tables are stored as Delta tables, enabling time travel for point-in-time correct training datasets — a critical concept for preventing data leakage in production models.
  • Model Serving on Serverless — New questions on deploying models to Databricks Model Serving in serverless mode — autoscaling to zero, pay-per-token pricing, and integration with Databricks Apps for business user-facing model endpoints.
Why This Matters for Accenture GTM

The ML Associate certification is the primary credential for Accenture consultants who participate in AI/ML delivery engagements on Databricks — not just data engineers building pipelines. As enterprise clients increasingly ask "who on the team is actually certified to build and deploy models on Databricks?", holding the ML Associate signals hands-on capability across the model lifecycle. It is also the recommended precursor to the GenAI Engineering Associate exam — making it part of a structured upskilling path for Accenture's 25,000-professional Databricks training commitment. For client development, presenting ML-certified data scientists builds confidence with CDOs and Chief AI Officers that Accenture can deliver production AI, not just platform setup.
