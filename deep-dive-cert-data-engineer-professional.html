Certification Overview

The Databricks Certified Data Engineer Professional is the apex credential in the data engineering track — designed for practitioners who architect, build, and operate production-grade Lakehouse pipelines at enterprise scale. Unlike the Associate exam, which tests foundational platform knowledge, the Professional exam demands deep technical mastery: query plan optimization, shuffle tuning, watermark strategies for streaming, and Delta Lake internals (transaction log, compaction algorithms, change data feed). As of 2026, the exam has been significantly strengthened with new questions on Unity Catalog advanced access patterns, Liquid Clustering versus Z-ordering trade-offs, and GenAI pipeline integration via Lakeflow.

Format: 120 minutes · ~60 questions · Scenario-based, multiple-choice · Recertification required every 2 years · Associate-level certification recommended as prerequisite.

2026 Exam Trends — What's New
  • Liquid Clustering is now testable — The 2026 GA release of Liquid Clustering is included in exam questions. Expect scenarios comparing ZORDER, OPTIMIZE, and Liquid Clustering for specific query pattern optimization.
  • Deletion Vectors for compliance — GDPR right-to-erasure patterns using Deletion Vectors (soft deletes without full rewrites) appear in 2026 exams, replacing older full-table-rewrite patterns.
  • Lakeflow AI-native pipelines — New questions on integrating AI functions (ai_query(), ai_summarize()) into DLT pipelines for unstructured data processing — reflecting Databricks' AI-first data engineering push.
  • Unity Catalog lineage in CI/CD — Exam scenarios now include Unity Catalog lineage APIs in production deployment pipelines, testing whether candidates can validate data dependencies before promotion.
