What is Unity Catalog?

Unity Catalog is Databricks' unified governance solution that provides centralized access control, auditing, lineage, and data discovery for all data and AI assets across an enterprise's Databricks deployments. Unlike traditional data catalogs that operate as passive metadata stores bolted on after the fact, Unity Catalog enforces policy at query time — meaning every access, transformation, and model inference is automatically governed and audited. Critically, Unity Catalog governs not just tables and files, but also ML models, AI agents, and external data connections — making it the world's first truly unified data and AI governance layer.

In 2026, Databricks open-sourced Unity Catalog, transforming it from a proprietary governance mechanism into an industry standard — while offering the managed version as a premium platform feature with extended enterprise capabilities.

Key Capabilities
Fine-Grained Access Control
Row-level and column-level security enforced transparently at query time — no application code changes required. Role-based access control (RBAC) with attribute-based policies. Dynamic data masking for PII columns by user group.
Automated Column-Level Lineage
Unity Catalog automatically traces the full lineage of every column — from source ingestion through transformations to downstream dashboards and ML models — without any manual annotation. Surfaced in the UI and queryable via API for compliance reporting.
Delta Sharing (Zero-Copy)
Share governed data with external partners, customers, or other cloud accounts without copying it. Delta Sharing is an open protocol — recipients can query shared tables using any tool that reads Delta, including Python, Spark, or Tableau.
Lakehouse Federation
Query external databases (MySQL, PostgreSQL, Snowflake, Redshift) directly through Unity Catalog's governance layer without data movement. Enables a unified query surface across the entire enterprise data estate.
AI-Powered Documentation
Unity Catalog uses AI to automatically generate table and column descriptions by analyzing data patterns and query history. Genie uses these descriptions to enable natural-language querying, dramatically reducing time-to-insight for business users.
Unity AI Gateway (2026)
Extended governance to AI agents: now governs Model Context Protocol (MCP) servers with end-to-end observability, cost attribution, and fine-grained permissions. Every agent call is logged, audited, and rate-limited — critical for enterprise AI governance.
